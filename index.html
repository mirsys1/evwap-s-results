💎 Part 3: 차등 투자 알고리즘 (R51-R80)

등액 투자(equal weight)가 아닌 신호 강도에 따라 자본을 차등 배분하는 알고리즘 평가.
각 거래의 가중치를 정규화하고 가중 수익률 계산. 같은 조합에서 weighting 방식만 변경하여 효과 비교.

10가지 가중치 함수

코드함수의도
EQUALw = 1등액 (baseline)
SCOREw = score신호 강도 비례
S5_A5w = s5 + a5VWAP 가속 강도
SCORE_SQw = score²강한 신호 강조 (집중)
INV_ATRw = 1 / ATR저변동성 우선 (안정)
SCORE_INV_ATRw = score / ATR강도 ÷ 변동성 (Sharpe형)
VWAP_DISTw = vwap_distVWAP 위 거리 비례
VOL_RATIOw = vol_ratio거래량 폭발 비례
COMBINEDw = score × vol_ratio / max(0.5, ATR)다중 요인 통합
KELLY_PROXYw = 0.1/0.3/0.5/1.0 by score 단계Kelly 분수 근사

💡 차등 투자 실전 적용 가이드

1. Position cap 강제: 단일 거래 ≤ 자본의 15% (max_pos 44% → 15%로 제한)
2. Score-based weighting 추천: w_i = max(0.5, score_i / mean_score)
3. Volatility-adjusted 보조: ATR 높은 종목은 weight ↓
4. 일일 총 노출 한도: 5종목 동시 보유 시 자본 100% 사용 (margin 0)
5. Pyramiding 금지: 같은 종목 추가 매수 X — Kelly의 가정 위반

🔑 핵심 통찰 5가지

  1. 시초 갭 ≤ 3% 필터가 단일 최강 — baseline의 갭상승 추격 손실을 거의 제거
  2. VWAP 연속 위 + 작은 익절 조합으로 승률 86%↑ 달성 (TRAIN→HOLD 안정)
  3. Score 가중치: 등액 대비 NET 1~3%p 추가 — 단 max_pos cap 필수
  4. VWAP 재돌파 / dynamic exit는 단일 신호로 효과 미미
  5. RSI 50-70 + MACD+ + R6 3중 필터는 신호 빈도 급감 (n=15) — 표본 부족
